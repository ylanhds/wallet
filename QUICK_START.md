**文件**: [`src/main.rs`](d:\projet\cargo\wallet-service\src\main.rs)

#### 新增内容：
```rust
// 添加 HTML 响应支持
use axum::response::Html;

// 主页路由处理函数
async fn index() -> Html<&'static str> {
    Html(include_str!("../index.html"))
}

// 添加路由
.route("/", get(index))
```

## 🔒 安全提醒

### ⚠️ 开发模式注意事项

当前配置为开发模式 (`ENVIRONMENT=dev`)：
- ✅ 会返回明文助记词和私钥
- ⚠️ **仅用于测试和学习**
- ❌ **不要用于生产环境**
- 🔐 **请妥善保管显示的敏感信息**

### 💡 生产环境建议

如果要部署到生产环境：
1. 设置 `ENVIRONMENT=prod`
2. 添加用户认证系统
3. 使用 HTTPS
4. 实施速率限制
5. 添加 CORS 策略
6. 使用环境变量管理密钥
